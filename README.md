RM-NPC immunotherapy analysis

This repository contains the R code used for the statistical analyses in patients with recurrent/metastatic nasopharyngeal carcinoma (RM-NPC).
The analyses include time-dependent ROC analysis, Inverse Probability of Treatment Weighting (IPTW), and weighted survival analyses.
Due to ethical restrictions, patient data are not publicly available. The data that support the findings of this study are available from the corresponding author upon reasonable request and with permission from the Ethics Committee of Sun Yat-sen University Cancer Center.
The R session information, including package versions, is provided in sessionInfo.txt.
